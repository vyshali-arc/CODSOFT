# Password Generator (Python)
This project is a secure password generator developed using Python.
It allows users to generate strong and random passwords by specifying
the desired password length and complexity.

## Features
- User-defined password length
- Option to include:
  - Uppercase letters (A-Z)
  - Lowercase letters (a-z)
  - Numbers (0-9)
  - Special characters
- Generates secure passwords using Python's `secrets` module
- Input validation and error handling

## Technologies Used
- Python
- secrets module
- string module
