# To-Do List Application (Python)
This is a GUI-based To-Do List application developed using Python and the Tkinter library.  
The application helps users manage their daily tasks by allowing them to add, update, and delete tasks easily.

## Features
- Add new tasks
- Update selected tasks
- Delete individual tasks
- Clear all tasks at once
- User-friendly graphical interface

## Technologies Used
- Python
- Tkinter (for GUI)

## How to Run
1. Make sure Python is installed on your system.
2. Open a terminal in the project folder.
3. Run the following command:
4. The To-Do List window will open on your desktop.

## Usage
- Enter a task in the input box and click **Add Task**.
- Select a task and click **Update Selected** to modify it.
- Select a task and click **Delete Task** to remove it.
- Click **Clear All** to remove all tasks.
